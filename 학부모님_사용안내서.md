## 학부모님을 위한 사용 안내서

---

### 🎯 우리 아이의 과학 성적, 이제 더욱 체계적으로 관리하세요!

목동에이원과학학원의 **온라인 성적관리 시스템**은 학생들의 학습 성과를 실시간으로 확인하고, 맞춤형 과제를 제공하여 효과적인 학습을 돕는 스마트한 학습 관리 도구입니다.

---

## ✨ 주요 특징

### 📱 편리한 온라인 접속
- **언제 어디서나** 스마트폰, 태블릿, PC로 접속 가능
- 별도의 앱 설치 불필요, 웹브라우저만 있으면 OK!

### 📊 실시간 성적 확인
- 테스트 제출 즉시 **자동 채점**
- 섹션별 정답률과 취약 영역을 **한눈에 파악**
- 누적 성적과 **성적 추이 그래프**로 학습 흐름 확인

### 🎯 맞춤형 과제 제공
- 오답 개수에 따라 **3단계 과제 자동 배정**
  - 📗 0-2개 오답: 시험지에 오답문제 정리해오기
  - 📙 3-4개 오답: 수업노트 필기 다시하고 오답문제 정리하기
  - 📕 5개 이상 오답: 동영상수업 내용복습이후고 수업노트필기 1번필기하고 오답문제들에 알가나뭍 적어오기

### 📈 학습 리포트 생성
- 원하는 기간의 **종합 학습 리포트** 생성
- 섹션별 정답률 분석
- 배정된 과제 현황 확인

## 👨‍🎓 학생 사용 방법

### 1️⃣ 로그인하기

```
1. 웹사이트 주소 접속: https://smart-test-weld.vercel.app
2. "학생 로그인" 페이지에서 정보 입력
   - 학생 ID (학원에서 부여)
   - 이름
   - 학년
3. "로그인" 버튼 클릭
```

> 💡 **Tip**: 한 번 로그인하면 자동으로 저장되어 다음에는 바로 대시보드로 이동합니다!

---

### 2️⃣ 테스트 응시하기

**📍 테스트 선택**
- 대시보드에서 "사용 가능한 테스트" 확인
- 응시할 테스트 선택

**✍️ OMR 답안 입력**
- 총 30문제 (각 부당 10문제씩)
- 1번~5번 중 정답 선택
- 모바일에서도 편리하게 입력 가능

**📤 제출 및 결과**
- "제출하기" 버튼으로 답안 제출
- **즉시 자동 채점** 후 결과 확인
- 섹션별 점수와 배정된 과제 확인

---

### 3️⃣ 성적 확인하기

**📊 대시보드**
```
✅ 누적 통계
- 총 응시 테스트 수
- 평균 점수
- 완료한 과제 수

📈 성적 추이
- 최근 테스트 점수를 그래프로 확인
- 학습 흐름을 한눈에 파악

🎯 배정된 과제
- 현재 해야 할 과제 목록
- 과제 난이도별 색상 구분
```

**📝 테스트 결과 상세**
```
- 섹션별 정답/오답 확인
- 틀린 문제의 정답 확인
- 배정된 과제 내용 확인
```

---

### 4️⃣ 학습 리포트 생성

**기간별 종합 분석**
```
1. 대시보드에서 "학습 리포트 생성" 클릭
2. 분석 기간 선택 (시작일 ~ 종료일)
3. "리포트 생성" 버튼 클릭

📋 리포트 내용
- 기간 내 테스트 성적 추이
- 섹션별 정답률 분석
- 배정된 과제 현황
- 학습 개선 포인트
```
