## 1. 시작하기

### 스마트 테스트란?
스마트 테스트는 학생들의 학습 진도를 확인하고 취약점을 분석해주는 온라인 테스트 시스템입니다.

### 주요 기능
- 온라인 테스트 응시
- 실시간 채점 및 결과 확인
- 단원별 성적 분석
- 맞춤형 추가 과제 제공
- 학습 리포트 생성

### 접속 방법
1. 웹 브라우저(Chrome, Safari, Edge 등)를 엽니다
2. 다음 주소로 접속합니다: **https://smart-test-weld.vercel.app**
3. 로그인 화면이 나타납니다

## 2. 로그인

### 로그인 방법

```
+----------------------------------+
|         스마트 테스트            |
|                                  |
|   학생 ID: [____________]        |
|                                  |
|   이름:    [____________]        |
|                                  |
|         [ 다음 ]                 |
+----------------------------------+
```

**단계별 안내:**

1. **학생 ID 입력**
   - 선생님께서 부여해 주신 학생 ID를 입력합니다
   - 예: `student001`, `kim2024` 등

2. **이름 입력**
   - 등록된 본인의 이름을 정확히 입력합니다
   - 예: `홍길동`

3. **[다음] 버튼 클릭**
   - 정보가 맞으면 대시보드로 이동합니다

### 로그인 오류 시

| 오류 메시지 | 원인 | 해결 방법 |
|------------|------|----------|
| "학생을 찾을 수 없습니다" | 학생 ID가 등록되지 않음 | 선생님께 문의하여 등록 확인 |
| "이름이 일치하지 않습니다" | 이름이 틀림 | 등록된 이름으로 다시 입력 |

### 로그인 유지
- 한번 로그인하면 브라우저를 닫아도 로그인이 유지됩니다
- 다른 기기에서 사용하려면 다시 로그인해야 합니다

## 3. 대시보드

로그인 후 가장 먼저 보이는 화면입니다.

### 화면 구성

```
+--------------------------------------------------+
|  안녕하세요, 홍길동님! (고등1학년)    [로그아웃]   |
+--------------------------------------------------+
|                                                   |
|  +-------------+  +-------------+  +------------+ |
|  | 총 테스트   |  | 평균 점수   |  | 최근 점수  | |
|  |     5회     |  |   85점      |  |   92점     | |
|  +-------------+  +-------------+  +------------+ |
|                                                   |
|  [ 성적 추이 그래프 ]                             |
|                                                   |
|  추가 과제 (최근 2주)                             |
|  +----------------------------------------------+ |
|  | 섹션 1 | 기본 | 교재 p.15~20 복습            | |
|  | 섹션 2 | 심화 | 추가 문제 풀이               | |
|  +----------------------------------------------+ |
|                                                   |
|  최근 테스트 결과                                 |
|  +----------------------------------------------+ |
|  | 수학 - 함수 테스트     | 92점 | 2024.01.15   | |
|  | 수학 - 방정식 테스트   | 78점 | 2024.01.10   | |
|  +----------------------------------------------+ |
|                                                   |
|  [ 리포트 보기 ]              [ 새 테스트 ]      |
+--------------------------------------------------+
```

### 각 영역 설명

#### 1) 상단 정보
- **학생 이름**: 로그인한 학생의 이름
- **학년**: 등록된 학년 정보
- **로그아웃 버튼**: 클릭하면 로그아웃됩니다

#### 2) 통계 카드
| 항목 | 설명 |
|------|------|
| 총 테스트 | 지금까지 응시한 테스트 총 횟수 |
| 평균 점수 | 모든 테스트의 평균 점수 |
| 최근 점수 | 가장 최근에 본 테스트 점수 |

#### 3) 성적 추이 그래프
- 최근 10회 테스트의 점수 변화를 보여줍니다
- 점수가 오르고 있는지 내리고 있는지 한눈에 확인할 수 있습니다

#### 4) 추가 과제 목록
- **최근 2주 이내** 테스트에서 받은 과제만 표시됩니다
- 과제 난이도에 따라 색상이 다릅니다:
  - 🟢 **기본(하)**: 0~2개 오답 - 기본 복습 과제
  - 🟠 **보충(중)**: 3~4개 오답 - 보충 학습 과제
  - 🔴 **심화(상)**: 5개 이상 오답 - 집중 학습 과제

#### 5) 최근 테스트 결과
- 응시한 테스트 목록이 최신순으로 표시됩니다
- **클릭하면** 해당 테스트의 상세 결과를 볼 수 있습니다
- 표시 정보: 과목명, 테스트명, 점수, 응시일

#### 6) 하단 버튼
- **[리포트 보기]**: 학습 분석 리포트를 생성합니다
- **[새 테스트]**: 새로운 테스트를 응시합니다

## 4. 테스트 응시하기

### 4.1 테스트 선택

대시보드에서 **[새 테스트]** 버튼을 클릭하면 테스트 선택 화면으로 이동합니다.

```
+--------------------------------------------------+
|  [ ← 대시보드로 돌아가기 ]                        |
+--------------------------------------------------+
|                                                   |
|  학년 필터:                                       |
|  [전체] [중1] [중2] [중3] [고1] [고2] [고3]       |
|                                                   |
|  이용 가능한 테스트                               |
|  +----------------------------------------------+ |
|  | 수학 | 함수 기초 테스트                      | |
|  | 섹션: 1차함수 ~ 2차함수                      | |
|  +----------------------------------------------+ |
|  | 수학 | 방정식 테스트                         | |
|  | 섹션: 일차방정식 ~ 이차방정식                | |
|  +----------------------------------------------+ |
|  | 영어 | 문법 기초 테스트                      | |
|  | 섹션: 시제 ~ 조동사                          | |
|  +----------------------------------------------+ |
+--------------------------------------------------+
```

**테스트 선택 방법:**

1. **학년 필터 사용** (선택사항)
   - 특정 학년의 테스트만 보려면 해당 학년 버튼을 클릭합니다
   - [전체]를 클릭하면 모든 테스트가 표시됩니다

2. **테스트 클릭**
   - 응시하고 싶은 테스트를 클릭합니다
   - 클릭하면 바로 OMR 입력 화면으로 이동합니다

### 4.2 OMR 답안 입력

테스트를 선택하면 OMR 답안 입력 화면이 나타납니다.

```
+--------------------------------------------------+
|  수학 - 함수 기초 테스트                          |
|  진행률: ████████░░░░░░░░ 20/30                  |
+--------------------------------------------------+
|                                                   |
|  섹션 1: 1차함수 (1~10번)                         |
|  +----------------------------------------------+ |
|  | 1번  | ① ● ③ ④ ⑤ |  (2번 선택됨)           | |
|  | 2번  | ① ② ③ ④ ⑤ |  (미선택)               | |
|  | 3번  | ① ② ● ④ ⑤ |  (3번 선택됨)           | |
|  | ...                                          | |
|  | 10번 | ① ② ③ ④ ⑤ |                         | |
|  +----------------------------------------------+ |
|                                                   |
|  섹션 2: 2차함수 (11~20번)                        |
|  +----------------------------------------------+ |
|  | 11번 | ① ② ③ ④ ⑤ |                         | |
|  | ...                                          | |
|  +----------------------------------------------+ |
|                                                   |
|  섹션 3: 함수의 활용 (21~30번)                    |
|  +----------------------------------------------+ |
|  | 21번 | ① ② ③ ④ ⑤ |                         | |
|  | ...                                          | |
|  +----------------------------------------------+ |
|                                                   |
|               [ 제출하기 ]                        |
+--------------------------------------------------+
```

**답안 입력 방법:**

1. **문제 번호 확인**
   - 총 30문제로 구성되어 있습니다
   - 3개의 섹션으로 나뉘어져 있습니다 (각 10문제)

2. **답 선택**
   - 각 문제의 ①②③④⑤ 중 하나를 클릭합니다
   - 선택된 답은 ●로 표시됩니다
   - 답을 바꾸려면 다른 번호를 클릭하면 됩니다

3. **진행률 확인**
   - 상단의 진행률 바에서 몇 문제를 풀었는지 확인할 수 있습니다
   - 예: "20/30"은 30문제 중 20문제 답을 입력했다는 의미

4. **제출하기**
   - 모든 답을 입력한 후 **[제출하기]** 버튼을 클릭합니다
   - 빈 답이 있어도 제출할 수 있습니다 (빈 답은 오답 처리)

### 주의사항
- 제출 후에는 답을 수정할 수 없습니다
- 제출 전에 모든 답을 확인해 주세요
- 인터넷 연결이 필요합니다

## 5. 결과 확인하기

테스트를 제출하면 자동으로 결과 페이지로 이동합니다.

### 5.1 결과 화면 구성

```
+--------------------------------------------------+
|  수학 | 고등1학년 | 함수 기초 테스트              |
|  2024년 1월 15일 오후 3:30                        |
+--------------------------------------------------+
|                                                   |
|          +------------------+                     |
|          |                  |                     |
|          |       92        |    훌륭해요! 🎉      |
|          |       점         |                     |
|          +------------------+                     |
|                                                   |
+--------------------------------------------------+
|                                                   |
|  빠른 통계                                        |
|  +-------------+  +-------------+  +------------+ |
|  | 정답        |  | 오답        |  | 총 문항    | |
|  |    27개     |  |    3개      |  |   30개     | |
|  +-------------+  +-------------+  +------------+ |
|                                                   |
+--------------------------------------------------+
```

### 5.2 점수별 메시지

| 점수 | 메시지 |
|------|--------|
| 90점 이상 | 훌륭해요! 🎉 |
| 70~89점 | 잘했어요! 👍 |
| 50~69점 | 힘내세요! 💪 |
| 50점 미만 | 열심히 해봐요! 📚 |

### 5.3 섹션별 성적 분석

```
+--------------------------------------------------+
|  파트별 성적 분석                                 |
+--------------------------------------------------+
|                                                   |
|  [도넛 차트]          섹션별 정답률               |
|                       +-----------------------+   |
|   정답 27             | 섹션 1: 1차함수       |   |
|   오답 3              | ██████████░░ 9/10     |   |
|                       | 섹션 2: 2차함수       |   |
|                       | ████████░░░░ 8/10     |   |
|                       | 섹션 3: 함수의 활용   |   |
|                       | ██████████████ 10/10  |   |
|                       +-----------------------+   |
+--------------------------------------------------+
```

- **도넛 차트**: 전체 정답/오답 비율을 시각적으로 보여줍니다
- **섹션별 프로그레스 바**: 각 섹션의 정답률을 막대로 표시합니다

### 5.4 응시 현황 및 순위

```
+--------------------------------------------------+
|  응시 현황                                        |
+--------------------------------------------------+
|  전체 응시: 25명  |  평균 점수: 78점             |
+--------------------------------------------------+
|  TOP 5 순위                                       |
|  +----------------------------------------------+ |
|  | 🥇 1위 | 홍*동 | 98점 |  ← (나)              | |
|  | 🥈 2위 | 김*수 | 95점 |                       | |
|  | 🥉 3위 | 이*영 | 92점 |                       | |
|  |    4위 | 박*철 | 90점 |                       | |
|  |    5위 | 최*희 | 88점 |                       | |
|  +----------------------------------------------+ |
+--------------------------------------------------+
```

- 같은 테스트를 본 학생들의 순위를 확인할 수 있습니다
- 개인정보 보호를 위해 이름 중간 글자는 *로 표시됩니다
- 본인의 순위는 "(나)"로 표시됩니다

### 5.5 상세 결과 및 과제

```
+--------------------------------------------------+
|  섹션 1: 1차함수 (1~10번)                         |
|  정답: 9개 / 10개                                 |
+--------------------------------------------------+
|  틀린 문제                                        |
|  +----------------------------------------------+ |
|  | 문제 | 내 답 | 정답 |                         | |
|  |   5  |   2   |  4   |                         | |
|  +----------------------------------------------+ |
|                                                   |
|  할당된 과제                                      |
|  +----------------------------------------------+ |
|  | 📗 기본 | 교재 p.25~30 복습하기              | |
|  +----------------------------------------------+ |
+--------------------------------------------------+
```

**표시 내용:**
- **틀린 문제**: 문제 번호, 내가 쓴 답, 실제 정답
- **할당된 과제**: 오답 수에 따라 자동으로 할당된 학습 과제

**과제 유형:**
| 아이콘 | 유형 | 오답 수 | 설명 |
|--------|------|---------|------|
| 📗 | 기본 | 0~2개 | 기본 복습 과제 |
| 📙 | 보충 | 3~4개 | 보충 학습 필요 |
| 📕 | 심화 | 5개+ | 집중 학습 필요 |

### 5.6 결과 출력하기

```
+--------------------------------------------------+
|        [ 결과 출력하기 ]  [ 메인으로 돌아가기 ]   |
+--------------------------------------------------+
```

- **[결과 출력하기]**: 결과를 프린터로 출력합니다
- **[메인으로 돌아가기]**: 대시보드로 돌아갑니다

## 6. 학습 리포트

대시보드에서 **[리포트 보기]** 버튼을 클릭하면 학습 분석 리포트를 볼 수 있습니다.

### 6.1 기간 선택

```
+--------------------------------------------------+
|  기간 선택                                        |
|  +--------------------+  [ 초기화 ]              |
|  | 📅 2024.01.01 ~    |                          |
|  |    2024.01.31      |                          |
|  +--------------------+                          |
+--------------------------------------------------+
```

- 캘린더에서 시작일과 종료일을 선택합니다
- **[초기화]** 버튼을 누르면 전체 기간의 데이터를 분석합니다

### 6.2 리포트 구성

```
+--------------------------------------------------+
|  홍길동 학습분석                                  |
|  2024.01.01 ~ 2024.01.31                         |
|  학습 수준: [상]                                  |
+--------------------------------------------------+
|                                                   |
|  요약                                             |
|  +-------------+  +-------------+  +------------+ |
|  | 📝 테스트   |  | 🎯 평균     |  | 📊 단원    | |
|  |    5회      |  |   85점      |  |    8개     | |
|  +-------------+  +-------------+  +------------+ |
|                                                   |
+--------------------------------------------------+
```

### 6.3 단원별 분석

```
+--------------------------------------------------+
|  단원별 분석                                      |
+--------------------------------------------------+
|                                                   |
|  [도넛 차트]          [레이더 차트]               |
|   단원별 정답 분포     단원별 정답률              |
|                                                   |
|  단원별 상세                                      |
|  +----------------------+  +--------------------+ |
|  | 1차함수              |  | 2차함수            | |
|  | 정답률: 90%          |  | 정답률: 75%        | |
|  | 정답: 18/20          |  | 정답: 15/20        | |
|  | 오답: 2개            |  | 오답: 5개          | |
|  | ████████████████░░░  |  | ██████████████░░░░ | |
|  +----------------------+  +--------------------+ |
+--------------------------------------------------+
```

**차트 설명:**
- **도넛 차트**: 어떤 단원에서 많이 맞추고 틀렸는지 비율로 보여줍니다
- **레이더 차트**: 각 단원별 정답률을 한눈에 비교할 수 있습니다

### 6.4 점수 추이

```
+--------------------------------------------------+
|  점수 추이                                        |
|  +----------------------------------------------+ |
|  |    100 ─┬─────────────────────────────────   | |
|  |         │              ●                     | |
|  |     80 ─┼──────●───────┼───●──────          | |
|  |         │   ●                   ●            | |
|  |     60 ─┼───────────────────────────         | |
|  |         │                                    | |
|  |      0 ─┴─────────────────────────────       | |
|  |        1/10  1/15  1/20  1/25  1/30          | |
|  +----------------------------------------------+ |
+--------------------------------------------------+
```

- 시간에 따른 점수 변화를 선 그래프로 보여줍니다
- 점수가 오르는지 내리는지 추세를 확인할 수 있습니다

### 6.5 취약 단원 분석

```
+--------------------------------------------------+
|  ⚠️ 취약 단원 분석                                |
+--------------------------------------------------+
|                                                   |
|  🎯 집중 필요: 2차함수                            |
|  정답률: 65% | 오답: 7개                          |
|                                                   |
|  📚 핵심 학습 내용:                               |
|  - 이차함수의 그래프 그리기                       |
|  - 꼭짓점과 축의 방정식 구하기                    |
|                                                   |
```

## 7. 자주 묻는 질문

### Q1. 학생 ID를 잊어버렸어요
**A:** 선생님께 문의하여 학생 ID를 다시 확인받으세요.

### Q2. 이름을 입력했는데 "이름이 일치하지 않습니다"라고 나와요
**A:** 선생님께서 등록하신 이름과 정확히 같게 입력해야 합니다. 띄어쓰기나 오타가 없는지 확인하세요.

### Q3. 테스트 도중에 브라우저가 닫혔어요
**A:** 다시 로그인하면 대시보드로 돌아갑니다. 아쉽지만 제출하지 않은 답안은 저장되지 않으므로 테스트를 다시 시작해야 합니다.

### Q4. 답을 잘못 입력하고 제출했어요
**A:** 제출 후에는 수정이 불가능합니다. 선생님께 상황을 말씀드리세요.

### Q5. 점수는 어떻게 계산되나요?
**A:** 총 30문제 중 맞은 개수를 비율로 계산합니다.
- 예: 27개 정답 → (27/30) × 100 = 90점

### Q6. 추가 과제는 어떻게 정해지나요?
**A:** 각 섹션별 오답 수에 따라 자동으로 할당됩니다:
- 0~2개 오답: 기본 복습 과제
- 3~4개 오답: 보충 학습 과제
- 5개 이상 오답: 심화 학습 과제

### Q7. 이전 테스트 결과를 다시 보고 싶어요
**A:** 대시보드의 "최근 테스트 결과" 목록에서 원하는 테스트를 클릭하면 상세 결과를 볼 수 있습니다.

### Q8. 다른 컴퓨터에서 로그인하면 기존 기록이 보이나요?
**A:** 네, 모든 테스트 기록은 서버에 저장되므로 어디서든 로그인하면 동일한 기록을 볼 수 있습니다.

### Q9. 핸드폰으로도 사용할 수 있나요?
**A:** 네, 모바일 브라우저에서도 사용할 수 있습니다. 화면이 자동으로 조절됩니다.

### Q10. 로그아웃은 어떻게 하나요?
**A:** 대시보드 상단의 **[로그아웃]** 버튼을 클릭하면 됩니다.
